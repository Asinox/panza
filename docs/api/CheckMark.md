`CheckMark` (component)
=======================

A basic CheckMark that can either be unchecked
or checked.
@composes Base, Icon

Props
-----

### `checkMarkIconName`

type: `string`
defaultValue: `'ios-checkmark-circle-outline'`


### `checkedColor`

type: `string`
defaultValue: `'positive'`


### `isChecked`

type: `bool`
defaultValue: `false`


### `size`

type: `number`
defaultValue: `30`


### `uncheckedColor`

type: `string`
defaultValue: `'light'`


### `uncheckedIconName`

type: `string`
defaultValue: `'ios-radio-button-off'`
