# Panza Components

Panza components are built with consideration to balancing techniques of  composition and the use of props to give you the best combination of flexibility and ease of use. Some components, like [InputRow](/docs/api/InputRow.md) do a lot, but also compose numerous smaller, simpler components. If you find yourself needing more flexibility, try using these smaller building blocks.

Typically, components compose the [Base](/docs/api/Base.md) or [TextBase](/docs/api/TextBase.md) components, and accept the same prop to style conversions.
