`SubNav` (component)
====================

Subnavigation, which displays three touchable text
options, typically used directly below the navigation
bar. Also typically used only on iOS.

Props
-----

### `backgroundColor`

type: `string`
defaultValue: `'white'`


### `borderBottom`

type: `bool`
defaultValue: `true`


### `options`

type: `arrayOf[object Object]`


### `textColor`

type: `string`
defaultValue: `'primary'`
