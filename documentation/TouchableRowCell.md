`TouchableRowCell` (component)
==============================

A touchable row cell container, typically used
rendering rows in a <ListView>.

Props
-----

### `disabled`

type: `bool`
defaultValue: `false`


### `height`

defaultValue: `60`


### `highlighted`

type: `func`
defaultValue: `function(){}`


### `onPress` (required)

type: `func`


### `showMore`

type: `bool`
defaultValue: `true`


### `underlayColor`

type: `string`
defaultValue: `'rgba(0,0,0,0.1)'`
