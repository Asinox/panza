`InputPicker` (component)
=========================

A touchable input field that expands (iOS) or
pops up (Android) to reveal a picker widget.

Props
-----

### `disabled`

type: `bool`


### `hasFocus` (required)

type: `bool`


### `label`

type: `string`


### `onRequestClose` (required)

type: `func`


### `onRequestFocus` (required)

type: `func`


### `showMore`

defaultValue: `true`


### `underlayColor`

type: `string`
defaultValue: `'rgba(0,0,0,0.1)'`


### `value` (required)

type: `string`
