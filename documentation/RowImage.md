`RowImage` (component)

An image stylized with sensible defaults
for display in a <TouchableRowCell />

-----

### `height`

type: `number`
defaultValue: `48`


### `rounded`

type: `bool`
defaultValue: `true`


### `source` (required)

type: `object`


### `width`

type: `number`
defaultValue: `48`
