`TouchableInput` (component)
============================

An input that responds to user touch, and is typically used
within the context of a navigator, or modal, to select
another value.

Props
-----

### `backgroundColor`

type: `string`
defaultValue: `'white'`


### `icon`

type: `node`


### `label` (required)

type: `string`


### `labelColor`

type: `string`
defaultValue: `'default'`


### `showMore`

type: `bool`
defaultValue: `false`


### `value`

type: `string`
